# 应用市场

>这是自己的第三个安卓项目，代码逻辑非常简单明了，涉及到大部分的安卓知识点，适合初识安卓的朋友学习。

##运行
> * app需要数据源，解压WebInfos.rar压缩包到手机根目录。
* 链接: http://pan.baidu.com/s/1eShzDGe 密码: trvm
* 然后安装源代码中的http服务器apk。直接启动就ok了。
